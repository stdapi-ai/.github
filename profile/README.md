# OpenAI-Compatible API Gateway for AWS Bedrock

We build enterprise-grade tools that make AWS AI accessible to everyone. Our flagship product, **stdapi.ai**, is a production-ready OpenAI-compatible API gateway that lets you deploy any OpenAI application on AWS Bedrock—no code changes required.

## 🚀 What is stdapi.ai?

**stdapi.ai** is a drop-in replacement for OpenAI SDK that connects your existing tools to 80+ AWS Bedrock models and AI services. Works with LangChain, Continue.dev, Open WebUI, n8n, and 1000+ OpenAI-compatible applications.

Simply change your API endpoint and instantly access Claude 4.6+, Nova 2, Llama 4, DeepSeek v3.2, and more—all while keeping your data in your AWS account.
